# Julia Köhn — Portfolio

Statische 7-Seiten-Website mit DE/EN-Umschalter (Standard: Deutsch).
Mountain-Sunset-Theme · Cormorant Garamond / DM Sans / DM Mono.

## Deployen (GitHub Pages)
1. Neues Repo (z.B. `portfolio`) anlegen.
2. Alle Dateien inkl. `assets/` ins Repo pushen.
3. Settings → Pages → Branch `main`, Ordner `/root`.
4. Fertig — die Seite liegt unter `https://<user>.github.io/portfolio/`.

Alternativ einfach `index.html` lokal im Browser öffnen.

Sprache wird über `localStorage` gemerkt und bleibt beim Blättern erhalten.
